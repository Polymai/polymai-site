Requirements

Bring your own AI and provider accounts

Polymai currently runs as a VS Code extension. You need VS Code installed, plus your own AI and provider accounts depending on what you build. You keep ownership of your OpenAI, ChatGPT/Codex, Supabase, GitHub, Stripe, Resend, and other provider accounts.

Recommended

OpenAI API key + ChatGPT account with Codex access. Best for full app builds, review loops, code changes, and complex SaaS workflows.

Supported

OpenAI API key only. Works for planning, generation, and simpler builds, but may need more manual review.

Experimental

Ollama/local models. Useful for testing and lightweight tasks, but not recommended yet for production-quality app generation.
